# Amazon Sales Dashboard (Power BI)

## 📊 Project Overview
This project analyzes Amazon sales data using Power BI to generate meaningful business insights.

## 🔍 Key Insights
- Total Revenue and Profit Trends
- Product-wise Performance Analysis
- Region-wise Sales Distribution
- Monthly Growth Trends

## 🛠 Tools Used
- Power BI
- Excel
- DAX
- Data Visualization
